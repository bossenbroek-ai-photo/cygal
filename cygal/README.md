# Cygal

The basic readme.

## Developing

```bash
 $ yarn dev
```

## Building

To create a production version of your app:

```bash
$ yarn build
```

## Storybook

Run storybook using,

```bash
$ yarn storybook
```

## Test

### Pre-requisite

Make sure to install playwright,

```bash
$ npx playwright install
```

### Run tests

To run the tests,

```bash
$ yarn test
```
